# PetCareScheduler

## Description
PetCareScheduler is a Java console application for managing pet registrations and scheduling appointments.  
It supports vet visits, grooming, and vaccinations. Users can display pets, view appointments, generate overdue reports, and save/load data from text files.

## Project Structure
Pet/
├─ classes/
├─ test/
└─ src/
├─ Appointment.java
├─ Pet.java
└─ PetCareScheduler.java

## How to Run
1. Make sure Java (JDK 11+) is installed.
2. Open terminal and navigate to the `src` folder.
3. Compile the Java files:
```bash
javac *.java
java PetCareScheduler
```

## Dependency
Only Java Standard Library (no external dependencies)
